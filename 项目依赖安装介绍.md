# vue-cli插件安装

> 最近的vue-cli

``` bash
# 首先下载node一路安装
# 然后看一下node和npm的版本
node -v
npm -v

# 安装yarn
npm install yarn -g

# 安装webpack
npm install webpack -g

# 查看webpack版本
webpack -v

# 安装vue-cli
npm install vue-cli -g

# 如果没有vue项目，初始化一个，根据提示操作即可，直到能看到页面为止
vue init webpack my-project

# 重头戏来了，看看需要安装什么插件

# 安装vuex
npm install vuex -save-dev

# scss
npm install node-sass sass sass-loader

#
```
